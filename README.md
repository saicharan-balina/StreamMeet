# StreamMeet

StreamMeet is a video conferencing web application that allows users to create and join meeting rooms for real-time communication.

## Features

* Video calling
* Audio calling
* Screen sharing
* Real-time chat
* Create and join meeting rooms
* Mute/unmute microphone
* Turn camera on/off

## Tech Stack

* React.js
* Node.js
* Express.js
* Socket.IO
* WebRTC
* MongoDB 

## Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/streammeet.git
cd  streammeet
```

### Install dependencies

Frontend

```bash
cd client
npm install
npm start
```

Backend

```bash
cd server
npm install
npm run dev
```

## Project Overview

The application uses WebRTC for peer-to-peer audio and video communication and Socket.IO for signaling and real-time messaging. Users can create meeting rooms and share room links with others to join video calls.
